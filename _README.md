# 클라이언트 개발자 포트폴리오 템플릿

`https://jinniepoo.github.io`의 페이지 구성과 어두운 포트폴리오 양식을 참고해 만든 GitHub Pages용 정적 템플릿입니다.

## 포함된 파일

- `index.html`: 홈, 소개, 프로젝트, 기술스택, 기타, 모달 구조
- `styles.css`: 원본과 유사한 다크 테마, 가로 스크롤 프로젝트 카드, 스킬 게이지
- `script.js`: 모바일 메뉴, 부드러운 스크롤, 스킬바 애니메이션, 프로젝트 모달

## 개인정보 제거 항목

- 실명
- 프로필 사진
- 학교 및 회사 로고
- 이메일
- 개인 GitHub / 블로그 / YouTube 링크
- 외부 문의 폼

## 사용 방법

내용을 본인 프로젝트에 맞게 수정한 뒤 GitHub 저장소 루트에 올리면 됩니다.
